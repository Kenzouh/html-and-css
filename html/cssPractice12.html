Relative

Relative Positioning: a type of position CSS property where it is positioned relative to where it should be placed.

width: 250px;
height: 550px;
background-color: #bf6aff;
position: Relative;
top: 0px;
left: 0px;

Absolute

Absolute Positioning: a type of CSS position property where it is positioned relative to the nearest ancestor.

width: 200px;
height: 250px;
background-color: #bf6aff;
position: Relative;
top: 0px;
left: 0px;

Sticky

Sticky Positioning: a type of CSS position value where it is positioned based on scroll position. Similar to Fixed Positioning but takes the whole width space at the initial position.

width: 250px;
height: 250px;
background-color: #b0ffda;
position: sticky;
top: 0px;
left: 100px;

Static

Static Positioning: a type of CSS position value where it is placed on the default positioning for an element. This is the default type of positioning!

width: 250px;
height: 250px;
background-color: #fbffd3;
position: static;

Fixed

Absolute positioning: a type of CSS position value where it is positioned relative to the viewport.

width:250px;
height: 200px;
background-color: #ff96b7;
position: fixed;
right: 0;
bottom: 0px;
